# NeuroArcade

NeuroArcade is a Java console-based arcade application featuring different mini games.

This project was developed as part of my Java Programming II course at Roosevelt University.

## Features
- Number Guesser
- Math Quiz
- Word Scramble
- Trivia Quiz
- Score tracking
- Recent games history

## Concepts Demonstrated
- Object-Oriented Programming
- Abstract classes and inheritance
- Data structures (ArrayList, Linked list)
- File I/O
- Command line interfaces
